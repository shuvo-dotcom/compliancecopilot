Everything you need. Nothing you don't.

Built for compliance teams who can't risk sensitive documents leaving the building.

Zero data egress

Your documents, credentials, and reports never leave your machine. Postgres, Redis, and MinIO all run locally in Docker.

Passkey auth only

No passwords. No OAuth. WebAuthn passkeys (Touch ID, Face ID, security key) — nothing sensitive stored in the database.

Any LLM provider

OpenAI, Anthropic, Gemini, Ollama (local), Groq, Mistral, Azure — via LiteLLM. Your API key lives in browser memory only.

Human approval queue

Every report goes through a human review step before it's finalised. Edit individual findings, approve or reject the whole report.

YAML frameworks

GDPR and EU AI Act ship out of the box. Add any framework by dropping a YAML file — no code changes needed.

One command setup

The macOS app handles everything. Or run docker compose up yourself — full stack in under a minute.

How it works

Six AI agents collaborate to produce a thorough gap report.

01

Upload

Drop a PDF, DOCX, or TXT policy document into the dashboard.

02

Chunk & Embed

The document is split into overlapping chunks and stored in pgvector for semantic search.

03

Retrieve

Relevant excerpts are matched to each GDPR or EU AI Act requirement.

04

Gap Analysis

Each requirement is checked — Compliant, Partial, or Missing — with citations from your document.

05

Risk Scoring

Findings are prioritised by severity and remediation urgency.

06

Human Review & Export

You review, edit findings, and approve — report is never finalised without human sign-off.

Install ComplianceCopilot

Free. Open source. Runs entirely on your Mac.

Step 1 — Prerequisites

🍺 Install Homebrew — brew.sh 🐳 Install Docker Desktop — docker.com

Step 2 — Install

brew tap shuvo-dotcom/compliancecopilot
brew install --cask compliancecopilot

Step 3 — First launch

  1. 1.Open ComplianceCopilot from Applications
  2. 2.If macOS shows a security warning, go to System Settings → Privacy & Security and click "Open Anyway"
  3. 3.The menu bar icon appears — the stack starts automatically

Updating

Run this anytime to get the latest version

brew upgrade --cask compliancecopilot
